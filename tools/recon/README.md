# Recon Tools

Tools for passive and active reconnaissance.

## Categories
- Subdomain enumeration
- DNS reconnaissance
- Web scraping / crawling
- Email harvesting
- Social media OSINT
- Certificate transparency

## Add Tools
Submit a PR adding the tool as a git submodule:
```bash
git submodule add https://github.com/owner/tool-name tools/recon/tool-name
```
