# Offensive Security Tools

Tools for penetration testing, exploitation, red team operations, and APT TTP simulation.

## Categories
- C2 frameworks (Havoc, Sliver)
- Windows infrastructure attacks (NetExec, Impacket, BloodHound, Responder)
- Web application testing (Nuclei, ffuf, Feroxbuster, Katana)
- Initial access simulation (GoPhish, Evilginx2)
- Credential tools (AuthBridge)
- API security (ReqReaper, RatRace)
- LLM red teaming (OmniFuzz-LLM)
