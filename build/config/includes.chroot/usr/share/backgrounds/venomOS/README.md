# VenomOS Wallpapers

Place wallpaper images here. The default wallpaper should be named:

  venomOS-default.png

Recommended specs:
- Resolution: 3840x2160 (4K, scales down cleanly)
- Format: PNG
- Style: Dark background (#0d0d0d), green (#00ff41) and red (#cc0000) accents
- Concept: serpent coiled around a target reticle, minimal/geometric

The wallpaper path is referenced in:
  config/includes.chroot/etc/skel/.config/xfce4/xfconf/xfce-perchannel-xml/xfce4-desktop.xml
